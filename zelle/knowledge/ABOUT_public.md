# About — Public Knowledge

## Portfolio Overview

This portfolio showcases three systems currently in active development:

1. SAFE — A risk assessment engine for financial markets (observer-only, research phase)
2. The Owl — An AI governance layer that enforces proposal-first change control
3. Cyber Agent — A security validation framework that tests AI defenses

Demos include real validation data and sample outputs. Internals are redacted. No financial advice is provided. No sensitive implementation details are exposed.

## How the Systems Connect

- The Owl governs AI actions through auditable approval flows
- Cyber Agent tests security — it audited The Owl and gave it the highest defense rating
- SAFE assesses market risk independently in observer mode

Each system is a standalone project. They demonstrate different aspects of AI governance, security, and risk assessment.
